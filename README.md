This is a comprehensive Maya scene inspection and processing tool that includes seven major functional modules: Scene Outliner, Model Topology, Model UV, Materials, Lighting and Rendering, Rigging, and Animation. The tool provides automated inspection and one-click processing features to help users quickly identify and resolve issues within the scene.

这是一个全面的Maya场景检查和处理工具，包含7大功能模块：场景大纲、模型拓补、模型UV、材质、灯光渲染、骨骼绑定和动画。工具提供自动化检查和一键处理功能，帮助用户快速发现和修复场景中的问题。
